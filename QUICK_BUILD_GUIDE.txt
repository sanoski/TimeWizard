================================
VRS TIME WIZARD - QUICK BUILD GUIDE
================================

WHAT IS THIS?
-------------
An Expo React Native offline timesheet app with local SQLite database.
Built with Expo SDK 51 (managed workflow).

PROJECT TYPE: Expo (NOT standard React Native CLI)
BUILD SYSTEM: EAS Build (Expo Application Services) - REQUIRED

CRITICAL INFO:
--------------
- Main app directory: /frontend (this is what you build)
- Backend directory: /backend (NOT needed for build - optional server)
- Database: expo-sqlite (native module, requires compilation)
- Cannot use standard "react-native run-android"
- Must use EAS Build or expo prebuild

FASTEST WAY TO BUILD APK:
-------------------------

1. Prerequisites:
   - Node.js 18+ installed
   - Install: npm install -g eas-cli
   - Create free Expo account at expo.dev

2. Build Commands:
   cd frontend
   npm install
   eas login
   eas build --platform android --profile preview

3. Wait 10-20 minutes for cloud build
4. Download APK from link provided
5. Install on Android: adb install app.apk

KEY FILES TO CHECK:
-------------------
- frontend/package.json (all dependencies)
- frontend/app.json (Expo config)
- frontend/eas.json (build config - create if missing)

DEPENDENCIES:
-------------
Main ones that require native compilation:
- expo-sqlite ~14.0.0 (LOCAL DATABASE - CRITICAL)
- expo-file-system (for backup)
- expo-router (navigation)
- @tamagui/config (UI)

IF BUILD FAILS:
---------------
1. Try: npm install, npx expo-doctor
2. Clear cache: rm -rf node_modules, npm install
3. Check Node version: node --version (must be 18+)
4. Verify Expo CLI: expo --version
5. Check BUILD_INSTRUCTIONS.md for detailed troubleshooting

TESTING THE APK:
----------------
After install, verify:
- App launches (may show migration screen first time)
- Can add hours in timesheet
- Data persists after restart
- Works in airplane mode (offline test)
- Export/import works

ALTERNATIVE (if EAS doesn't work):
----------------------------------
Local build (more complex):
1. Install Android Studio + SDK
2. cd frontend
3. npx expo prebuild --platform android
4. cd android
5. ./gradlew assembleRelease
6. APK in: android/app/build/outputs/apk/release/

IMPORTANT NOTES:
----------------
- This is OFFLINE-FIRST app, backend not needed
- Web version won't work (SQLite native only)
- First build takes longest (20+ min)
- EAS Build is FREE for basic usage
- App needs Android 6.0+ (API 23+)

COMMON ERRORS:
--------------
"expo-sqlite not found" → Use EAS Build
"Module not resolved" → Run npm install
"Gradle failed" → Try ./gradlew clean
"Build timeout" → Normal, just wait longer
